About

The Lynx assistant is currently able to support seven different requests.

  • atomic_weight.get: Given an element or symbol, the assistant responds with the average atomic mass of the specific element.
  • element.get: Given a symbol, the assistant responds with the name of the element tht the symbol represents.
  • group.check: Given an element or symbol and a group number or group name (ie noble gases), the assistant responds with whether or not the given element is found in the given group. If the element does not appear in the given group, it will also tell users which group it is found in.
  • group.get: Given an element, the assistant responds with which group the element is found in, as well as the name of the group (ie noble gases).
  • period.check: This request is similar to the group.check request, but this request checks for periods rather than groups.
  • period.get: Given an element, the assistant responds with which period the element is found in.
  • symbol.get: Given an element, the assistant responds with which symbol represents the element.

Contribute

The Lynx Assistant is built by the community on GitHub. It is built in JavaScript for the web and licensed under the GPL-3.0. Please visit the GitHub repository if any of the following apply to you. You:

  • Have any issues or bugs to bring up
  • Are experienced with web design
  • Are experienced with JavaScript
  • Are experienced with iOS or Android development
  • Have any suggestions for the assistant
